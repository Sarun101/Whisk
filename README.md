##Description 
The app is intialized with a login screen with allows the user to login as either a Customer or Shopkeeper
#Customer
The user will see a Menu item with options to add or delete number of items from specific food items or type it manually, they will also get a recommendation feed
#Shopkeeper
The user here will get food orders labelled with the user name, with multiple users ordering they are met with multiple order lists identified by name and id
